Information Security Laboratory

Information Security Laboratory (IS-521) is a postgraduate hands-on course offered at GSIS (Graduate School of Information Security), KAIST, Korea.

The primary goal of this course is to "learn by doing". Every student in this course will work on a series of hands-on exercises using Git and GitHub as their primary tools.

This course started in spring 2017 by prof. Sang Kil Cha.
